### Setup
```shell
$ git clone git@github.com:vlad-litvinenko/that-simple.git
$ git submodule update --init
$ cd lib/libdaisy
$ make
$ cd ../.. 
$ make clean; make
```

### Upload
```shell
$ make program-dfu
```
